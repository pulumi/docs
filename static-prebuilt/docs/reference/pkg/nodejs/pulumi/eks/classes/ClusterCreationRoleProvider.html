Class ClusterCreationRoleProvider

ClusterCreationRoleProvider is a component that wraps creating a role provider that can be passed to new eks.Cluster("test", { creationRoleProvider: ... }). This can be used to provide a specific role to use for the creation of the EKS cluster different from the role being used to run the Pulumi deployment.
