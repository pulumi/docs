Class ConfigGroup

Creates a set of Kubernetes resources from Kubernetes YAML text. The YAML text may be supplied using any of the following ConfigGroupArgs:

  1. Using a list of filenames: Files = new[] { &quot;foo.yaml&quot;, &quot;bar.yaml&quot; }
  2. Using a list of file patterns: Files = new[] { &quot;foo/*.yaml&quot;, &quot;bar/*.yaml&quot; }
  3. Using literal strings containing YAML: Yaml = new[] { &quot;(LITERAL YAML HERE)&quot;, &quot;(MORE YAML)&quot; }
  4. Any combination of files, patterns, or YAML strings.
