Class Deployment

Provides an API Gateway Deployment.

Note: Depends on having aws.apigateway.Integration inside your rest api (which in turn depends on aws.apigateway.Method). To avoid race conditions you might need to add an explicit depends_on = [&quot;aws_api_gateway_integration.name&quot;].
