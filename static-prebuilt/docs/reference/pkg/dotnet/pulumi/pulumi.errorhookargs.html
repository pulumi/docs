Class ErrorHookArgs

ErrorHookArgs represents the arguments passed to an error hook. Depending on the failed operation, only some of the new/old inputs/outputs are set. 

| Failed Operation | old_inputs | new_inputs | old_outputs |
| ---------------- | ---------- | ---------- | ----------- |
| create           |            | ✓          |             |
| update           | ✓          | ✓          | ✓           |
| delete           | ✓          |            | ✓           |
