Class Index

Fields

The fields supported by this index. The last field entry is always for the field path 'name'. If, on creation, 'name' was not specified as the last field, it will be added automatically with the same direction as that of the last field defined. If the final field in a composite index is not directional, the 'name' will be ordered '"ASCENDING"' (unless explicitly specified otherwise).
