Combines all the Input<T> values in inputs and combines them all into a single Output<T> with an containing all their underlying values. Format(FormattableString)

Takes in a with potential Input<T>s or Output<T> in the 'placeholder holes'. Conceptually, this method unwraps all the underlying values in the holes, combines them appropriately with the string, and produces an Output<T> containing the final result.

If any of the Input<T>s or Output<T>s are not known, the final result will be not known. Similarly, if any of the Input<T>s or Output<T>s are secrets, then the final result will be a secret.
