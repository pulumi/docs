# Pulumipus mascot assets

**For current Pulumipus mascot artwork, go to [brand.pulumi.com/identity/mascot/](https://brand.pulumi.com/identity/mascot/).**

The older mascot illustrations that used to live here have been moved to
[`legacy/`](./legacy/). They are deprecated and kept only because a few existing
pages and blog posts still reference them — they are **not** the current or
canonical mascot assets.

- **Need a Pulumipus for general use** (a deck, a partner site, an external
  doc)? Get it from
  [brand.pulumi.com/identity/mascot/](https://brand.pulumi.com/identity/mascot/) —
  don't reuse the files in `legacy/`.
- **Adding a file here?** Fine when the codebase actually references it to render
  the site (e.g. the current mascot artwork, once finalized, belongs here). This
  just isn't a general store for mascot assets or the canonical source — that's
  brand.pulumi.com.
