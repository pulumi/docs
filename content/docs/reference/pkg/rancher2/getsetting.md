Use this data source to retrieve information about a Rancher v2 setting.

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as rancher2 from "@pulumi/rancher2";

const server_image = pulumi.output(rancher2.getSetting({
    name: "server-image",
}, { async: true }));
```
