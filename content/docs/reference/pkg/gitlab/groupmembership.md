This resource allows you to add a user to an existing group.

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gitlab from "@pulumi/gitlab";

const test = new gitlab.GroupMembership("test", {
    accessLevel: "guest",
    expiresAt: "2020-12-31",
    groupId: "12345",
    userId: 1337,
});
```
