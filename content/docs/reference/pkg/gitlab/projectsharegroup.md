This resource allows you to share a project with a group

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gitlab from "@pulumi/gitlab";

const test = new gitlab.ProjectShareGroup("test", {
    accessLevel: "guest",
    groupId: 1337,
    projectId: "12345",
});
```
