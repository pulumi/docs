{{% examples %}}
## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gcp from "@pulumi/gcp";

const my_forwarding_rule = gcp.compute.getForwardingRule({
    name: "forwarding-rule-us-east1",

```
