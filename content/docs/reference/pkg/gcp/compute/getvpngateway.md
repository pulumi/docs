Get a VPN gateway within GCE from its name.

{{% examples %}}
## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as gcp from "@pulumi/gcp";

const my_vpn_gateway = pulumi.output(gcp.compute.getVPNGateway({
    name: "vpn-gateway-us-east1",
}, { async: true }));
```
