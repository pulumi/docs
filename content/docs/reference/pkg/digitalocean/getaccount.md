Get information on your DigitalOcean account.

## Example Usage

Get the account:

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as digitalocean from "@pulumi/digitalocean";

const example = pulumi.output(digitalocean.getAccount({ async: true }));
```
