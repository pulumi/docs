            title="Optional">

    <dt class="property-optional"
            title="Optional">
        <span>Load<wbr>Distribution</span>
        <span class="property-indicator"></span>
        <span class="property-type">string?</span>
    </dt>
    <dd>{{% md %}}Specifies the load balancing distribution type to be used by the Load Balancer. Possible values are: `Default` ‚Äì The load balancer is configured to use a 5 tuple hash to map traffic to available servers. `SourceIP` ‚Äì The load balancer is configured to use a 2 tuple hash to map traffic to available servers. `SourceIPProtocol` ‚Äì The load balancer is configured to use a 3 tuple hash to map traffic to available servers. Also known as Session Persistence, where  the options are called `None`, `Client IP` and `Client IP and Protocol` respectively.
{{% /md %}}</dd>

    <dt class="property-required"
            title="Required">
