{{% examples %}}
## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as openstack from "@pulumi/openstack";

const network = openstack.networking.getNetwork({
    name: "tf_test_network",
});
```
