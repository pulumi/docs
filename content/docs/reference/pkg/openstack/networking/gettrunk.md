Use this data source to get the ID of an available OpenStack trunk.

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as openstack from "@pulumi/openstack";

const trunk1 = pulumi.output(openstack.networking.getTrunk({
    name: "trunk_1",
}, { async: true }));
```
