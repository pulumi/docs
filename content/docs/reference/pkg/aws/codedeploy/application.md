Provides a CodeDeploy application to be used as a basis for deployments

{{% examples %}}
## Example Usage

{{% example %}}
### ECS Application

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as aws from "@pulumi/aws";

const example = new aws.codedeploy.Application("example", {
    computePlatform: "ECS",
});
```
