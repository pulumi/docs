`aws.wafregional.WebAcl` Retrieves a WAF Regional Web ACL Resource Id.

## Example Usage

```typescript

import * as aws from "@pulumi/aws";

const example = aws.wafregional.getWebAcl({
    name: "tfWAFRegionalWebACL",
```
