Use this data source to fetch information about a Quantum Ledger Database.

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as aws from "@pulumi/aws";

const example = aws.qldb.getLedger({
    name: "an_example_ledger",
});
```
