`aws.waf.Rule` Retrieves a WAF Rule Resource Id.

## Example Usage

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as aws from "@pulumi/aws";

const example = aws.waf.getRule({
    name: "tfWAFRule",
});
```
