Use this data source to retrieve the collaborators for a given repository.

{{% examples %}}
## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as okta from "@pulumi/okta";

const example = pulumi.output(okta.app.getApp({
    label: "Example App",
}, { async: true }));
```
