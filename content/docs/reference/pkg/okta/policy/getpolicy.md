## Example Usage
{{% example %}}

```typescript
import * as pulumi from "@pulumi/pulumi";
import * as okta from "@pulumi/okta";

const example = okta.policy.getPolicy({
    name: "Password Policy Example",
    type: "PASSWORD",
});
```
