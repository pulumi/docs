distribution

distribution: pulumi.Output<string | undefined>
  • Defined in cloudwatch/logSubscriptionFilter.ts:37

The method used to distribute log data to the destination. By default log data is grouped by log stream, but the grouping can be set to random for a more even distribution. This property is only applicable when the destination is an Amazon Kinesis stream. Valid values are "Random" and "ByLogStream".

Sourced from https://www.terraform.io/docs/providers/aws/r/cloudwatch_log_subscription_filter.html#distribution.
