Install [Go](https://golang.org/doc/install).

> **Note:** Pulumi requires a [supported version of Go](https://go.dev/doc/devel/release#policy) — this typically refers to the two most recent major releases. Note that Go calls 1.20, 1.21, etc. major releases, unlike semantic versioning. If you're using Linux, your distribution may not provide an up to date version of the Go compiler. To check what version of Go you have installed, use: `go version`.
