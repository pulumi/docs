Install [Java 11 or later](https://www.oracle.com/java/technologies/downloads) and [Apache Maven 3.6.1 or later](https://maven.apache.org/install.html).

> **Note:** Pulumi will need the `java`, `javac`, and `mvn` executables in order to build and run your Pulumi Java application. Ensure that these executables can be found on your path after installation.
