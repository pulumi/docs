Cloud Engineering Days

Opening Keynote - Founder/CEO Joe Duffy and CTO Luke Hoban

Joe Duffy and Luke Hoban discuss the newest innovations in infrastructure as code and modern applications and how pioneering engineering organizations use these techniques to ship faster, more reliably, and at a greater scale.

Discover what's new:
  • Pulumi Deployments: groundbreaking advancements in Deployments-as-a-Service technology by Pulumi
  • Pulumi Templates: new templates providing architecture-specific starting points for the most common infrastructure as code use cases on AWS, Azure, and Google Cloud.
  • Pulumi YAML GA: now Generally Available as part of its 1.0 release, enabling everyone in an organization to use Pulumi to manage cloud infrastructure.
  • Project Level Configuration: new support across all Pulumi languages makes it easier to manage complex configurations as cloud projects scale up.
